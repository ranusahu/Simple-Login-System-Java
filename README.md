# Simple Login System (Java)

This is a simple Java console-based login system application.

## Features
- User login using username and password
- Validates credentials
- Displays login success or failure message

## Concepts Used
- String comparison
- If-else conditions
- Scanner class
- Console input/output

## How to Run
1. Compile the program:
   javac LoginSystem.java
2. Run the program:
   java LoginSystem
